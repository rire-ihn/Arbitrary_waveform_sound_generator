# arbitrary_waveform_sound_generator

csvファイルからwavファイルを作ります。これにより、任意の波形の音を作ることができます。

## 実行の方法

Pandas, Numpy, Scipyのインストールが必要です。

実行コマンド：
```bash
python main.py
```

指示に従い、読み込むcsvファイル、長さ、周波数を入力すると、wavファイルを出力します。

## 読み込むcsvファイルの要件

- 2列で、区切りはカンマです。行数に指定はありません。

- 1列目がR、2列目がLの値です。

- 値は-1から1までの値である必要があります。
